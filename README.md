# Budget Control System

ระบบ Budget Control เบื้องต้นด้วย Flask + Bootstrap

## Features
- Login / Logout
- Dashboard
- Budget Input Form
- Budget List
- Contract Expiry Tracking
- Upload Excel Import
- SQLite Database
